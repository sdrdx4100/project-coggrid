# Project CogGrid

『メダロット・navi』のロボトルに着想を得た、Godot製の非営利ファンゲーム／戦闘システム試作です。

## 現在のプロトタイプ

- 9×9グリッドと4基のジェネレーター
- 推進値順の行動とターン開始時のAP供給
- パーツ行動のAPを先に支払い、残りAPで移動
- 頭部・右腕・左腕・脚部の4部位と部位破壊
- リーダー頭部の破壊による勝敗
- 余剰APのMFへの変換
- ジェネレーター占領による相手チームのAP減少
- 図形だけで描画する機体と簡易敵AI

## 必要環境

- Godot Engine 4.7.1 Standard

## 起動

Godotでこのフォルダーの `project.godot` を開き、メインシーンを実行します。

操作はマウスのみです。青チームの手番で行動パーツを選択し、盤面上の移動先、続いて攻撃対象をクリックします。赤チームは自動行動します。
