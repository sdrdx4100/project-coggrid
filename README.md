# Project CogGrid

『メダロット・navi』のロボトルに着想を得た、Godot製の非営利ファンゲーム／戦闘システム試作です。

## 現在のプロトタイプ

- タイトル画面、新規開始、コンティニュー
- 図形タイルで構成したフィールドと主人公の4方向移動
- NPC会話からロボトルを開始し、終了後にフィールドへ復帰
- RPGメニューでのチーム全機セッティング
- フィールド座標・勝利数・所持品・編成のセーブ／ロード
- 経験値とメダルレベル、微量の成功補正、レベル別AI・MF候補解放
- 9×9グリッドと4基のジェネレーター
- 推進値順の行動とターン開始時のAP供給
- パーツ行動のAPを先に支払い、残りAPで移動
- 頭部・右腕・左腕・脚部の4部位と部位破壊
- 独立したパーツカタログ、所持パーツ、4スロットへの装着・交換
- 装着パーツによる技・装甲・推進・回避・防御・図形外見の変化
- 行動クラスと光学・火薬などの攻撃系統を分離した拡張可能な行動定義
- 初回行動前に利用できるセッティング画面
- 成功値・回避値・防御値による命中判定とダメージ軽減
- リーダー頭部の破壊による勝敗
- 余剰APのMFへの変換
- ジェネレーター占領による相手チームのAP減少
- 図形だけで描画する機体と簡易敵AI

## 必要環境

- Godot Engine 4.7.1 Standard

## 起動

Godotでこのフォルダーの `project.godot` を開き、メインシーンを実行します。

フィールドでは矢印キーまたはWASDで移動、EnterまたはSpaceで会話、MまたはEscapeでメニューを開きます。中央研究所の研究員へ話しかけるとロボトルを開始できます。

ロボトルはマウスで操作します。行動パーツを選択し、盤面上の移動先、続いて攻撃対象をクリックします。赤チームは自動行動します。
